### Como usar o projeto

- Clonar o projeto

  [Git Clone](https://github.com/cabraldasilvac/GitHubActions.git)

```bash
    git clone https://github.com/cabraldasilvac/GitHubActions.git
```

- Instalar as dependências do projeto

```bash
      yarn install
```

- Rodar o projeto

```bash
      yarn dev
```

- Rodar o teste

```bash
        yarn jest
```
